Questions & Answers

No Question Answer
1 Give two examples of information that can be put in meta tags. Both the description of the website & the keywords for the website can be put into the metatags. the advantage for these is that they increase the search rankings of the website if used correctly.
2 What are div tags used for in developing web pages? Div tags are used to divide the page up into segments, grouping other HTML elements together and styling them as a group.
3 Give two examples of how HTML has changed from HTML 1.0 to HTML5 HTML 5 has multimedia support allowing the addition of elements such as videos without the need of javascript. With HTML 5 came the addition of the header and footer elements allowing better structuring of the page.
4 Explain what an HTML attribute is and give an example. A HTML attribute provides additional information about an HTML element. An example is "Class" which allows the user to group elements together and style them as one.
5 What is the difference between a Class and an ID? A class can be used by multiple elements while an ID can only be used by one element within the HTML page.
6 Explain pseudo-elements affect on hyperlinks: A:hover A:link A:visited A:active A:hover - Affects the hyperlink when the mouse hovers over it.
A:link - Affects the hyperlink when it is not visited.
A:visited - Affects the hyperlink when it is visited.
A:active - Affects the hyperlink when it is clicked.
7 Give one advantage and one disadvantage of inline styling. Advantage - It allows you to override any external styles for specific elements.
Disadvantage - It can be difficult to maintain and can make the code hard to read.
8 Explain the difference between margins and padding in CSS. Margins affect the space around the outside of the element while padding affects the space around the inside of the element.
